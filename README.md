# BPSR-ModuleIntegrater

ブループロトコル：スターレゾナンス (BPSR) 用のスクリーンショット・リアルタイム画面キャプチャを基に、プラグイン（モジュール）の効果等のパラメータを自動解析・抽出するツールです。

## 注意
開発中のプロトタイプです。
画像解析の精度が環境によって不安定です。

## 主な機能

1. **画面撮影・画像読み込み**
   - 画面の一部を切り取って解析対象とする「画面撮影」機能。
   - すでに保存してあるスクリーンショットの「画像ファイル読み込み」機能。
2. **リアルタイム自動監視**
   - 一度キャプチャ範囲を指定すれば、その範囲の変更を一定間隔で自動検知して次々とリストに追加していく監視機能。
3. **パラメータの自動解析**
   - 自動文字認識（OCR・パターンマッチ）を用いて、「攻撃速度」「HP吸収」などの各ステータス項目とその数値を自動抽出。
   - うまく認識されない場合の「調整スライダー（ブロックサイズ・しきい値調整）」と、誤認識してしまった場合の手動修正機能（リストのダブルクリック）。
4. **CSV/JSON保存**
   - 蓄積した解析結果のリストを、後から表計算ソフト等で扱いやすい「CSV」や「JSON」形式にまとめてエクスポート可能。

## 使い方・操作説明

### 1. 手動での解析（単発）
1. ツールを起動後、**「画像を開く」**ボタンで既存のスクショを選ぶか、**「画面撮影」**ボタンで画面上の解析したい領域（パラメータのリスト部分）をドラッグして囲みます。
2. 対象画像が読み込まれると、左側にプレビュー（文字等を認識するための緑枠・青枠）が表示されます。
   - ※ 緑の枠が1行1行を正しく囲めていない場合は、左側の「ブロックサイズ (分割調整)」スライダーを動かして調整してください。
   - ※ うまく認識されない場合は「サイズ倍率」や「しきい値」スライダーを調整してください。
3. **「解析実行 (手動)」**ボタンを押すと、右上のパネルに抽出された結果（項目名と値）が表示されます。
4. 結果に間違いがあれば、値の部分を**ダブルクリックして直接修正**できます。
5. 問題なければ**「↓ リストに追加 ↓」**ボタンを押し、右下のストック領域に一旦保存します。

### 2. リアルタイム監視（連続解析）
ゲーム内でマウスをスクロールしたり複数ページにまたがる一覧を次々とデータ化したい場合に便利です。
1. **「画面撮影」**ボタンを押し、解析したい領域をドラッグして指定します。
2. **「監視 開始」**ボタンを押します。ボタンが赤くなり監視がスタートします。
3. そのままゲーム内でページをめくったり数値を切り替えたりすると、内容の変化を検知して自動で次々と右下のストック領域に追加されていきます（画面のチラつきによる重複分は自動で除外されます）。
4. 終わる時は**「■ 停止」**ボタンを押してください。

### 3. 一括エクスポート
1. 必要なパラメーターをリストにすべてストック（追加）した状態にします。
2. 右下にある**「まとめて保存 (CSV/JSON)」**ボタンをクリックします。
3. 保存先と保存ファイルの種類（CSVまたはJSON）を選んで保存します。出力されたファイルはExcel等で管理可能です。
